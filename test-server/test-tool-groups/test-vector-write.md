> âš ď¸Ź **AGENT TRAP â€” `isError: true` rules for tools with `outputSchema`**: The MCP SDK uses `isError` to decide whether to validate `structuredContent` against the `outputSchema`. >
> | Response | `isError: true` | SDK behavior | Verdict |

> | `success: false` | **Absent** | Validates error against success schema â†’ fails | âťŚ Bug â€” raw `-32602` |
>
> **TL;DR**: `isError: true` on errors, absent on successes. Do NOT blanket-add or blanket-remove it.
