# Configuration (`config/`)

This directory contains example configuration files and templates for setting up the `db-mcp` environment.

## Contents

- **`db-mcp.keycloak.example.json`**: An example configuration payload demonstrating how to configure Keycloak identity providers or similar identity management integrations.

_Note: Actual configuration files or `.env` files should be placed at the project root and are generally ignored by Git to prevent leaking sensitive credentials._
