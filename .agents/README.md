# AI Agent Workflows (`.agents/`)

This directory contains configuration files and standard operating procedure (SOP) workflows for AI coding agents operating within this repository.

## Structure

- **`workflows/`**: Markdown-based workflow guides providing step-by-step instructions for specific agent tasks (e.g., `audit-code-quality.md`, `pr-review.md`, `issue-triage.md`).

These workflows are automatically discovered by agent tools (like Google Antigravity) and are invoked to ensure consistency, quality, and adherence to project rules across automated coding and auditing tasks.
